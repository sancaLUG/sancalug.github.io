Installfest

A Installfest é o evento, ou festival, em que são feitas instalações de distribuições GNU/Linux em computadores de pessoas interessadas no sistema operacional. Daí o nome “Installfest”: do inglês installation festival. Ele é uma das portas de entrada para novos usuários no universo FLOSS.

Para quem vai participar, seguem algumas orientações:

  • É essencial que os participantes realizem o backup (ou cópia de segurança) de seus dados e a desfragmentação de seus discos rígidos previamente.
    • Para o Windows® 10 a desfragmentação pode ser feita seguindo os passos desse link
  • Ler o Código de conduta e o Termo de consentimento previamente. Contactar o LUG em caso de dúvida.

Distribuição e Ambiente de Desktop sugeridos ao público iniciante

Para possibilitar o formato e a elaboração de conteúdo de apoio, a organização concordou que seria benéfico ter uma distribuição sugerida para os iniciantes. Os critérios que essa distribuição deve atender são:

  • estabilidade;
  • compatibilidade com hardwares;
  • amigável/user-frendly;
  • possuir tutoriais e helps na internet; e
  • know-how prévio dos organizadores na distribuição

Após discussões da organização, a distribuição que obteve o melhor balanço entre os critérios foi o Ubuntu. Ele é uma distro mais conhecida do publico em geral, promovendo uma certa familiaridade ao participante. Além disso, há um aplicativo nativo para baixar outros aplicativos e atualizar o sistema. Em sua última versão, o ambiente de desktop usado no Ubuntu é o GNOME. Há uma recomendação de capacidade mínima para os PCs.

Formato

As instalações no dia serão feitas por ordem de chegada, individualmente ou em duplas, a partir de uma lista de espera com nome dos interessados na instalação. Uma cópia de uma distribuição GNU/Linux será gravada no pendrive dos interessados.

Quando chegar a vez do interessado, com esse pendrive será inicializado o sistema para instrução de uso antes da instalação. A instrução de uso será realizada parte por um voluntário da organização do evento e parte pelo interessado e tem como objetivo colocar o novo usuário em contato com o sistema antes da instalação e fazer um teste de compatibilidade de hardware. O teste da compatibilidade ocorrerá naturalmente conforme for seguindo a instrução de uso. A instrução seguirá as partes mais pertinentes do roteiro do mini-curso à dupla. O voluntário irá alternar a apresentação entre os computadores da dupla para que cada interessado tenha a oportunidade de aprender com a “mão-na-massa”.

Após a familiarização do usuário com o novo sistema, segue-se com a instalação. Antes da instalação, o Termo de consentimento deverá ser lido e assinado. Temos uma versão resumida também. Para a instalação, algumas configurações deverão ser feitas no Windows: gerenciar espaço no disco rígido para o novo sistema e permitir boot fora do Windows (essa última provavelmente feita para conseguir rodar o pendrive). O Windows será reiniciado para verificar se a alteração de disco não gerou problemas. O pendrive com Linux será iniciado novamente para fazer a instalação do sistema no disco rígido.

Ao final será entregue um formulário de feedback para que o sancaLUG possa continuamente melhorar os eventos.

Para voluntários

O sancaLUG está sempre buscando voluntários para ajudar com demonstrações, instalações e pós-instalações. Se você tem interesse em ajudar como voluntário auxiliando os participantes na installfest, por favor veja mais informações em Installfest para os voluntários e entre em contato conosco no nosso grupo do Telegram.

Nossa base de conhecimento de solução de problemas comuns pode ser consultada durante a installfest. Esteja a vontade para comunicar um novo problema ou solução.
